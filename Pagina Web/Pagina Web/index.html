Diseño web




HTML significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto).
Es el lenguaje base de la web, el que se usa para estructurar el contenido de las páginas.




CSS


CSS significa Cascading Style Sheets (Hojas de Estilo en Cascada).
Es el lenguaje que se usa junto con HTML para definir cómo se ve una página web: colores, tamaños, posiciones, fuentes, bordes, animaciones, etc.




https


HTTPS es el acrónimo de HyperText Transfer Protocol Secure, es decir, Protocolo Seguro de Transferencia de Hipertexto. Es la versión segura de HTTP (el protocolo que usan los navegadores para comunicarse con los servidores web)




Hipertexto


El hipertexto es un sistema de escritura y lectura digital donde el texto puede contener enlaces (links) que llevan a otros textos, imágenes, videos u otros recursos.




UX


Es la experiencia general de un usuario, se refiere a cómo se siente una persona al interactuar con un producto o servicio digital




Prototipado


El prototipado es el proceso de crear una versión preliminar (simplificada) de un producto digital o físico para:


  • Visualizar ideas.
  • Probar funciones.
  • Recibir retroalimentación antes de construir la versión final.

En diseño digital, se aplica mucho en UX/UI para simular cómo se verá y funcionará una app o sitio web.

Diagrama de flujo DOM


Un diagrama de flujo es una representación gráfica de los pasos que conforman un proceso, mostrando cómo se conectan entre sí de manera secuencial. Se utiliza para entender, analizar o comunicar cómo se realiza una tarea o cómo funciona un sistema.




Etiquetas

Las HTML tags (etiquetas HTML) son los elementos básicos del lenguaje HTML que se utilizan para estructurar y dar formato al contenido de una página web.

Características principales:
Están encerradas entre los signos < >.
Normalmente vienen en pares: una etiqueta de aperturay una de cierre. La de cierre lleva una barra inclinada / para indicar el final del contenido.
Dentro pueden llevar atributos, que añaden información extra o modifican el comportamiento de la etiqueta.




Conferencia IA


Mi experiencia con las conferencias de la inteligencia artificial organizadas por la UACJ es principalmente negativa, debido a que para empezar no pude acceder a la primera, debido a la terrible organización por parte de la institución, y tengo constancia de que en la segunda ocasión, pese a que si logré entrar, muchos alumnos se quedaron afuera y se les negó la entrada.
En cuanto a los temas, considero que no estuvieron a la altura de lo que una conferencia como esta representa, tocaron los temas de manera muy general, y considero que cualquier persona que esté un poco informado de las novedades sobre la inteligencia artificial tiene conocimiento de gran parte de lo que se impartio. Sin embargo, creo que si hubo algunos buenos puntos que pueden ser de utilidad para los alumnos que no estuvieran tan informados sobre el tema, además de que sirvió como plataforma para que los alumnos se expresaran sus opiniones sobre la inteligencia artificial.




Propuesta para identificar IA



Mi propuesta para identificar media hecha por IA es agregar una línea de color al exterior del archivo, video o imagen, de forma sutil, y evitando que se distorsione el contenido o composición del diseño
